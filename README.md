# User Management API

Proyek ini merupakan bagian dari tugas Asynchronous 3 â€“ Back-End Development Part 3, yang bertujuan untuk membangun RESTful API menggunakan Node.js, Express, dan PostgreSQL. API ini menyediakan fitur registrasi, login dengan JWT, pengelolaan data pengguna (lihat semua user, update profil sendiri, hapus akun sendiri), serta upload avatar menggunakan Cloudinary. Sistem sudah dilengkapi validasi email dan password, penyimpanan waktu updated_at setiap kali profil diubah, serta middleware keamanan seperti CORS, Helmet, dan verifyToken. Struktur proyek telah dipisahkan menjadi folder controllers, models, routes, dan middleware agar lebih rapi dan mudah dikelola. Semua konfigurasi sensitif disimpan dalam file .env.
